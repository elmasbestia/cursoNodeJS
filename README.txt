Este repositorio sirve para soportar el curso relámpago de Node JS

La idea de este curso es dar las nociones para comenzar a desarrollar con Node JS en una forma sucinta.

Todos los archivos contenidos en este directorio están sujetos a modificación y, sobretodo, a colaboración por parte de los participantes.

El contenido actual es:
README.txt			Este archivo

Taller NodeJS.txt		Soporte para el dictado del Taller

Instalación			Notas acerca de la instalación de Node JS

TN001_Hola-Mundo.js		Ejemplo del programa básico en Node JS
TN002_Estructura-Rafa.js	Estructura típica de un aplicativo en Node JS según Rafa

Archivos propios del ambiente de Node JS:
package.json		Registro de los datos técnicos concernientes al ambiente de ejecución
node_modules		Librerías usadas por los aplicativos en Node JS
